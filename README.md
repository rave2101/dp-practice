# Dynamic Programming Practice

C++ solutions to classic Dynamic Programming problems, following [Striver's DP Playlist](https://www.youtube.com/playlist?list=PLgUwDviBIf0qUlt5H_kiKYaNSqJ81PMMY).

Each folder contains individual problem folders with `solution.cpp` and a `README.md` explaining the approach, recurrence, traces, and bugs encountered.

---

## How to Compile & Run

```bash
g++ -std=c++17 -o solution solution.cpp && ./solution
```
